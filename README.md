# codemind-ai

An AI-powered coding assistant built in Python, originally based on the [Build your own Claude Code](https://codecrafters.io/challenges/claude-code) challenge by [CodeCrafters](https://codecrafters.io).

This project uses Large Language Models (LLMs) to understand code and perform actions through tool calls. It implements an LLM-powered coding assistant with support for HTTP RESTful APIs, OpenAI-compatible tool calling, agent loops, and multi-tool integration.
